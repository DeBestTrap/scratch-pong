# scratch-pong
A super simple game of pong made from scratch.

Materials needed:
<ol>
  <li>1x Arduino Nano</li>
  <li>1x 128x32 OLED Display</li>
  <li>4x Pushbuttons</li>
  <li>4x 220 Ohm Resistors</li>
  <li>Jumper Wires</li>
</ol>
